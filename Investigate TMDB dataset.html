def check_col_dependency(col):
    mean_val = tmbd[col].mean()
    low = tmbd.query('{} < {}'.format(col, mean_val))
    high = tmbd.query('{} >= {}'.format(col,mean_val))
    size = low.shape[0] + high.shape[0] == tmbd.shape[0]
    flag = low['revenue_adj'].mean() < high['revenue_adj'].mean()
    final_val = 'column ' + col + ' size maintained: ' + str(size) + '\n' + 'Is ' + col + ' associated with high revenue earnings? : ' + str(flag) +'\n \n' + 'Associated supporting scatter plots are as below: \n'
    print(final_val)
    tmbd.plot(x = col , y="revenue_adj", kind="scatter");
    if(flag == True):
        effect = 'Positive'
    else:
        effect = 'Negative'            
    plt.title(effect + ' effect of ' + col + ' on revenue earned by movies \n')  
    plt.grid(True)

Conclusions¶

  • Year 2014 topped in maximum number of movie released. 684 movies were released this year.
  • People prefer adventure|action|thriller & adventure|fantasy|action movies more.
  • People prefer adventure movies most.
  • High earning movies have longer runtime than average, receive high votes and gain more popularity.
  • Budget of movies do not impact on revenue earnings

Limitations:¶

  • The analysis is limited to the provided dataset. We have worked with 'release_date','genres','popularity', 'budget_adj','vote_average','runtime', 'revenue_adj' variables mainly, present in TMDB movies dataset.
  • We are uncertain that every release between 1971-2070 is listed in TMDB movies dataset.
  • 140 movies' data was dropped due to missing values in variables under consideration. Although, they could potentially exhibit unintentional bias towards the relation being analysed.
  • No normalisation/exchange rate/currency conversion is considered during analysis
